C语言从入门到精通 Day 1

我的第一行
/#include<stdio.h>/

  1. include 指令
    这个语句的的功能是进行有关的预处理操作,include称为文件包含命令,后面尖括号的内容称为头部文件

  2. 空行
    空行 . 空格 . 跳格并不是影响程序,有了这些空格规范输入程序代码才能变得自然美观

  3. main函数声明
    这是声明mian函数为一个返回值,是整型函数.其中int称为关键字

  4. 函数体
    #include<stdio.h> //头文件
    int main() //函数声明,函数头
    {
    printf("hello,world!");//输出要显示的字符串,括号内容则为字符串
    return 0;//这行语句使main函数终止运行,并向操作系统返回一个整型常量0.
    }
