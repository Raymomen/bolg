for循环练习

倒叙 100 - 1 循环

代码如下:
include <stdio.h>
int main( )
{
int i ;
for( i = 100;i >= 1; i--)
printf("%d\n",i);
return 0;
}

注释

int i 写入变量"i"到内存
for( i = 100;i >= 1; i--) for循环 变量"i"=100; "i" 大于等于1; 变量"i"每一次循环减1;

  • < >表示导入系统库文件 " "则表示导入自定义文件
  • include 导入文件
  • std C语言自带的标准库之一
  • io input & output 标准输入输出库
  • .h 头文件
